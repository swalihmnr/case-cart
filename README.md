# CaseCart

CaseCart is a full-stack e-commerce web application focused on selling stylish and protective mobile phone cases. The platform allows users to explore a wide variety of mobile case designs for different smartphone models and purchase them through a smooth online shopping experience.

The application includes user authentication, product browsing, cart management, coupon discounts, and order processing. An admin panel is also available for managing products, users, orders, and promotional coupons.

The project is built using Node.js and Express.js for the backend, MongoDB for the database, and EJS for server-side rendering. The system follows an MVC architecture to maintain clean code organization and scalability.

## Key Features

* User registration and secure authentication
* Product listing with categories and filtering
* Shopping cart and wishlist functionality
* Coupon and discount system
* Order management and order history
* Secure checkout and payment handling
* Admin dashboard for product and order management
* Session-based authentication and authorization
* Image upload and product management
* Responsive UI for better user experience

## Architecture

The application follows an MVC (Model–View–Controller) architecture to maintain clean code separation and scalability. The backend is built as a RESTful service that communicates with the MongoDB database and serves dynamic views using EJS.

## Purpose of the Project

CaseCart was developed as a practical backend engineering project to demonstrate real-world e-commerce functionality, including authentication systems, coupon logic, order processing, and production deployment.
